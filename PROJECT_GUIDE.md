## Общая информация

«Сбор данных» — это Telegram Mini App для нутрициолога, предназначенная для сбора замеров тела и фотографий прогресса. Пользователь открывает Web App из Telegram-бота, заполняет замеры (вес, грудь, пупок, живот, бёдра, ноги, руки), опционально загружает до 3 фото, и данные отправляются в Leadteh через serverless API.

**Аккаунт Leadteh:** rb257034.leadteh.ru
**Webhook:** `https://rb729467.leadteh.ru/inner_webhook/0f32708a-4738-425e-903a-986db0842534`

## Описание файлов

### index.html
Основная страница приложения с формой замеров тела. Содержит:
- 7 числовых полей замеров (вес, грудь, пупок, живот, бёдра, ноги, руки)
- 3 опциональных поля загрузки фото с превью
- Сжатие фото на клиенте (canvas → JPEG, max 800px, quality 0.6)
- Валидацию всех полей
- Интеграцию с Telegram Web App API
- Отправку данных на серверный эндпоинт `/api/submit`

### offer.html, privacy.html, data-processing.html
Страницы документов (не используются в текущей форме, оставлены на случай необходимости):
- Каждая страница содержит текст соответствующего документа
- Кнопка «Назад» для возврата к форме
- Поддержка темы Telegram Web App

### api/submit.js
Vercel Serverless Function:
- Принимает POST-запросы с замерами и фото
- Валидирует 7 числовых полей (должны быть числа > 0)
- Принимает 3 опциональных фото (base64 строки)
- Отправляет данные в Leadteh webhook
- Возвращает результат в формате JSON

### vercel.json
Конфигурация деплоя на Vercel.

## Настройка интеграции с Leadteh

### Шаг 1: Создание бота в Telegram

1. Создайте бота через @BotFather
2. Получите токен бота
3. Настройте Web App через BotFather (Menu Button URL)

### Шаг 2: Настройка Leadteh

1. Войдите в аккаунт `rb257034.leadteh.ru`
2. Перейдите в **Настройки → Webhooks**
3. Webhook уже создан: `fe02b6a2-14d6-46c3-836f-5fb8292d0e4f`
4. Создайте сценарий обработки данных

### Шаг 3: Настройка переменных в Leadteh

Доступные переменные после получения webhook:
- `{{weight}}` — вес (кг)
- `{{chest}}` — грудь (см)
- `{{navel}}` — пупок (см)
- `{{stomach}}` — живот (см)
- `{{hips}}` — бёдра (см)
- `{{legs}}` — ноги (см)
- `{{arms}}` — руки (см)
- `{{photo_1}}` — фото 1 (base64 JPEG)
- `{{photo_2}}` — фото 2 (base64 JPEG)
- `{{photo_3}}` — фото 3 (base64 JPEG)
- `{{telegram_id}}` — ID пользователя в Telegram
- `{{source}}` — источник (telegram-webapp-data-collection)
- `{{submission_date}}` — дата отправки
- `{{ts}}` — timestamp

## Тестирование интеграции

1. Пользователь должен написать `/start` боту (создание контакта в Leadteh)
2. Открыть Web App через Telegram бота
3. Заполнить все 7 полей замеров корректными числами
4. Опционально загрузить фото
5. Нажать «Отправить замеры»
6. Проверить в Leadteh, что данные получены
